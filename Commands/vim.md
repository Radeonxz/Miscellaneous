### How to exit

#### Enter insert mode

```
i
```

#### Save changes

```
Esc + :w
```

#### Save and quit Vim

```
Esc + :wq
```

or

```
Esc + ZZ
```

#### Quit Vim and discarding all changes

```
Esc + :q!
```

#### Quit(a warning is printed if a modified file has not been saved)

```
:q
```

#### Copy a line

```
yy
```

#### Delete a line

```
dd
```
