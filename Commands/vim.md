### How to exit

#### Save changes
```
Esc + :w
```

#### Save and quit Vim

```
Esc + :wq
```
or
```
Esc + ZZ
```

#### Quit Vim and discarding all changes

```
Esc + :q!
```

#### Quit(a warning is printed if a modified file has not been saved)

```
:q
```

#### Save workspace and quit the editor (same as :wq)

```
zz
```
